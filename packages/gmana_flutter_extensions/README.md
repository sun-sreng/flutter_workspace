# gmana_flutter_extensions

Focused Flutter extension methods and small services for the Gmana ecosystem.

```dart
import 'package:gmana_flutter_extensions/gmana_flutter_extensions.dart';
```

## Includes

| Area | APIs |
| --- | --- |
| Color | `ColorExt`, `StringColorExtension`, `ColorService` |
| Responsive layout | `Breakpoint`, `Breakpoints`, `BreakpointUtils`, `ResponsiveContext` |
| Build context | `ContextExt` |
| Theme mode | `ThemeModeExt`, `ThemeModeStringExt`, `ThemeModeService` |
| Serialization | `IconDataExt`, `IconDataSerialization` |
| Time | `TimeOfDayExtensions` |

Use `gmana_flutter_extensions` directly when you only need helpers and do not
need the widgets or design tokens from `gmana_flutter`.
